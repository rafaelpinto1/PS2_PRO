# PS2_PRO
Advanced PS2 motherboard reengineering: Modchip installation, SATA/SSD adaptation for OPL (Open PS2 Loader), and HDMI/RetroGEM video output modernization. Aims to eliminate the optical drive, reduce loading times by up to 70%, and extend hardware lifespan.
