# wLaunchELF

`BOOT.ELF`: build de desenvolvimento do wLaunchELF (sucessor do uLaunchELF), usado como carregador inicial no pendrive pro modo "mass" do Matrix Infinity.

Não é meu, peguei direto de: https://github.com/ps2homebrew/wLaunchELF/releases (release "latest", build automática de 2026-07-19, commit d4ce420). Licença AFL 2.0, código-fonte original no mesmo repositório.

Origem do projeto (uLaunchELF): EP + dlanor + vários colaboradores da comunidade PS2. Mantido hoje pela organização ps2homebrew, com colaboradores atuais incluindo AKuHAK, uyjulian, sp193, rickgaiser, fjtrujy, entre outros (lista completa nos commits do repositório).

sha256: `2a82e4415b22964961948bc5829296e24d184ace0640e3c3bd4729109e283471`

Vai solto na raiz do pendrive de teste (não numa pasta), é a convenção que o modchip espera pra dar boot automático.
