# Firmware/homebrew usado no projeto

Binários de terceiros (homebrew PS2 já existente) usados como estão, cada um com o crédito e a licença na própria pasta:

- `wlaunchelf/`: gerenciador de arquivos/carregador, usado como `BOOT.ELF` no pendrive pro modo "mass" do Matrix Infinity.
- `opl/`: Open PS2 Loader, o loader de jogos em si.

Nada aqui foi escrito por mim, é software homebrew já pronto da comunidade PS2. Os créditos e licenças de cada um estão no README de cada subpasta.
