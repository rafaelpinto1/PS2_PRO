Peguei a versão beta em vez da estável (v1.1.0) de propósito, pra testar.

## Como vai no pendrive

```
/OPL/OPNPS2LD.ELF
```

As pastas de config/tema (`CFG`, `LNG`, `THM`, `ART`, etc.) o próprio OPL cria sozinho na primeira vez que abre, não precisa criar na mão.
